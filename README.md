## Collaboration GitHub

La collaboration a été simulée sur un seul poste de travail à l’aide de 2 comptes GitHub.
Les contributions ont été réalisées sur des branches distinctes, puis intégrées via des Pull Requests.
Bien que certains commits puissent inclure plusieurs fichiers, le workflow GitHub
(branches, Pull Requests, merge et résolution de conflits) est correctement respecté.
